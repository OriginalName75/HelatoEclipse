>>> clear : supprime l'historique des commandes

AFFICHER :

>>> afficher TABLE : Affiche tous les éléments de la table
>>> afficher TABLE attribut='a': Affiche les éléments de la table dont les attributs valent a
ex:
>>> afficher module : Affiche tous les modules
>>> afficher uv : Affiche tous les modules
>>> afficher groupe : Affiche tous les modules
>>> afficher module uv='1.1' :Affiche les modules qui appartiennent à l'UV '1.1'
>>> afficher personne sexe = homme :Affiche les personnes qui sont des hommes

AJOUTER:


>>> ajouter TABLE attribut='a': Ajouter les éléments dans la table avec les attributs valent a
ex:
>>> ajouter personne nom = zhu prenom = lei login = jules12345 mdp = 'jules1' sexe = homme type = eleve numero = 06254534578 promotion = 2017 adresse = 'SECRET' lieu_naissance= 'China france' date_naissance=12/05/1988 mail=jules@ensta.fr : Ajoute une personne zhu lei avec toutes ses informations
>>> ajouter groupe 'sp2a' : Ajoute un groupe 'sp2a'
>>> ajouter uv '4.9' : Ajoute une UV '4.9'
>>> ajouter module 'langage c' uv='4.9' : Ajoute un module 'langage C' dans l'UV '4.9'

MODIFIER :


>>> modifier TABLE attribut='a' nouveau attribut='b' : Modifie les elements d'une table qui ont les attributs avec les nouvelles valeurs
>>> modifier TABLE attribut='b' : Modifie les elements d'une table avec les nouvelles valeurs
ex:
>>> modifier personne lieu_naissance='Brest' nouveau lieu_naissance='jules' : Modifie les personnez qui ont les attributs 'a' avec la nouvelle valeur 'b'
>>> modifier module 'langage c' nom='Langage C++' uv='2.3' groupe='RAK' : Modifie le module 'langage C' avec les nouvelles valeurs
>>> modifier uv nom='4.9' nouveau='9.4' : Modifie l'UV '4.9' à '9.4'
>>> modifier groupe nom='CI2Apourtester' nouveau='CI2Ateste' : Modifie le groupe 'CI2Apourtester' à 'CI2Ateste'

SUPPRIMER :


>>> supprimer TABLE atribut='a': Supprimer les éléments qui valent a
ex:
>>> supprimer module 'Langage C++' : Supprimer la module 'Langage C++'
>>> supprimer groupe 'CI2Ateste' : Supprimer le groupe 'CI2Ateste'
>>> supprimer uv '9.4' : Supprimer l'UV '9.4'
